書くためのデジタルツール：断片からドキュメントへ

第5回　人文学とデジタル技術（2026）

🎯

本章の到達点

  • ●学術的文章作成の5ステップ（断片→整理→統合→ドキュメント化→共有）
  • ●各ステップで活用するツールの使い分け

内容

1

断片をつくる

紙のメモ・ノート・カード、Evernote 等。アイデアの種を集める段階。

2

断片を整理する

Obsidian：Markdown、リンク・バックリンク、タグ、MOC、Evergreen Notes によるネットワーク的整理。

3

断片を統合する

Scrivener：アウトライン管理、コルクボード、資料参照、分割ビュー、コンパイル機能による長文執筆支援。

4

ドキュメントをつくる

主なドキュメント作成ツールの比較（Notion / Word / Google Docs / LaTeX（Overleaf）の特徴と使い分け）。読みやすいドキュメント作成のポイント（見出し・段落・箇条書き・スタイル・保守性）。

5

ファイルを共有する

PDF 化と命名規則、体裁確認（見出し、段落、箇条書き、余白、統一感）。道具に振り回されず、思考と言葉を主役として執筆環境を設計する。

📝

授業時間外学習（チェックポイント）

研究テーマに関する「断片」を最低5件作成し、出典・日付を添えて整理する。Obsidian でノート2〜3件作成しリンク付け。文章の骨子をアウトラインで作成。Notion / Word / Google Docs / LaTeX のうち1つで短い文書を作り PDF 化。今後の執筆フローに取り入れる点と注意点を記述する。
